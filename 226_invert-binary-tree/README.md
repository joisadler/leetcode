Invert a binary tree.

**Example:**

Input:
<pre>
     4
   /   \
  2     7
 / \   / \
1   3 6   9
</pre>
Output:
<pre>
     4
   /   \
  7     2
 / \   / \
9   6 3   1
</pre>
